The IOTA SDK has a convenient
[`TransactionOption`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount)
called [`allowMicroAmount`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount) to help you easily
send amount below the minimum
[storage deposit](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/storage-deposit/), or microtransaction.

<TipFaucet />

## Code Example

The following example will:

1. Create a wallet.
2. Get Alice's account which was [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Create the outputs for a microtransaction.
4. Send the microtransaction using the
   [`Account.send(SendParams, TransactionOption?)` function](../../references/nodejs/classes/Account/#send) with
   the [`allowMicroAmount`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount) `TransactionOption` set as `true`.
