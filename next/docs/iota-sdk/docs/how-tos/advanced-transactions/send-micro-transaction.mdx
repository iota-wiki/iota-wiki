Any transaction with an amount that is below the minimum
[storage deposit](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/storage-deposit/) is considered a
microtransaction.

The IOTA SDK has a convenient
[`TransactionOption`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount)
called [`allowMicroAmount`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount) to help you easily
send a microtransaction.

## Code Example

<Tabs groupId="language" queryString>

<TabItem value="rust" label="Rust">

The following example will:

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html).
2. Get Alice's [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html) which was
   [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Sync the account with the node to get the latest transactions by calling the [`Account.sync()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.sync)
   function.
4. Send a micro transaction using the
   [`Account.send()` function](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.send) with
   the [`allow_micro_amount`](../../references/nodejs/interfaces/TransactionOptions/#allowmicroamount)
   [`TransactionOption`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.TransactionOptions.html).
   set as `true`.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/advanced_transactions/send_micro_transaction.rs
```

</TabItem>
