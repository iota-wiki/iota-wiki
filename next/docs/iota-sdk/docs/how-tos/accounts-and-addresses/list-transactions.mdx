This guide will show you how to list all transactions of an [`Account`](../../../references/nodejs/classes/Account/).

:::tip Sync Options

If you want to list incoming transactions you need to specify that in your `sync` call.
Transactions for outputs that you synced before with other sync options, will not be requested later.
Also, only transactions that weren't pruned on the point of syncing will be listed.

:::tip

## Code Example

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

The following example will:

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html).
2. Get Alice's [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html) which was
[created in the first guide](./create-account.mdx).
3. Get all the [`Transactions`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/types/struct.Transaction.html)
related to the `Account` by calling the [`Account.transactions()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.AccountInner.html#method.transactions)
function.
4. Get all the incoming `Transactions` related to the `Account` by calling the
[`Account.incoming_transactions()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.AccountInner.html#method.incoming_transactions) function.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/accounts_and_addresses/list_transactions.rs
```
