An [`Account`](../../../references/nodejs/classes/Account/) is a fundamental component of the protocol that allows users
to manage their digital assets and interact with the network. It serves as a container for holding tokens and other
assets, enabling users to send and receive transactions, and engage in various activities within the IOTA ecosystem.

## Code Example

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

The following example will:
1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html).
2. Use it to store the [Stronghold](https://wiki.iota.org/stronghold.rs/welcome) mnemonic.
3. Create an [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html) with the alias
"Alice" by calling the [`Wallet.create_account()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html#method.create_account)
function.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/accounts_and_addresses/create_account.rs
```
