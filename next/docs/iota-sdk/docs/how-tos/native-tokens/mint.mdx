You may want to increase the circulating native token supply. To do so, you will need to call the
`Account.mint_native_token(token_id, mint_amount, options)` function.

## Code Example

The following example will:

1. Create a wallet.
2. Get Alice's account which was [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Get the account's balance.
4. Mint 10 native tokens of the first ID.
