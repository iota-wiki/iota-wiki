Minting an [NFT](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/tokenization/#non-fungible-tokens-nfts)
refers to creating a unique and indivisible digital asset on [the Tangle](https://wiki.iota.org/shimmer/learn/tangle/).
When you mint an NFT, you usually associate a digital file, such as artwork, music, or video, with a specific token on
the Tangle, establishing its authenticity, ownership, and uniqueness. This process involves assigning
[metadata](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#metadata)
to the NFT, including details about the creator
(or [issuer](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#issuer)), the asset's
description, and any additional information relevant to its value or characteristics.

## Code Example

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

The following example will:

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html).
2. Get Alice's [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html) which was
   [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Sync the account with the node to get the latest transactions by calling the [`Account.sync()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.sync)
   function.
4. Define the `sender_address` as the first [`Address`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/address/enum.Address.html)
   in the [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html).
5. Create a new [`MintNftParams`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/struct.MintNftParams.html) and set its
   values. Note that these include setting the [sender](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#sender)
   and the [issuer](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#issuer).
6. Mint the NFTs using the [`Account.mint_nfts()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.mint_nfts)
   function.
7. Build the outputs for the transaction to mint another NFT using the [`NftOutputBuilder`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/nft/struct.NftOutputBuilder.html).
8. Send the transaction using the [`Account.send_outputs()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.send_outputs)
   function.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/nfts/mint_nft.rs
```
