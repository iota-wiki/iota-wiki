Once you have [signed data with Ed25519](sign-ed25519.mdx), you can use the
[`Utils.verifyEd25519Signature(signature, message)`](../../references/nodejs/classes/Utils/#verifyed25519signature)
function to verify that the data hasn't been modified by any third-party.
