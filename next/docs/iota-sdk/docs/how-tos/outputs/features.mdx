The Stardust protocol introduced
[multiple new output features](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/)
that allow you to add [metadata](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#metadata),
a [sender](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#sender)
(or [issuer](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#issuer) in the case of
[NFTs](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/tokenization/#non-fungible-tokens-nfts)) or [tag](https://wiki.iota.org/shimmer/develop/explanations/what-is-stardust/output-features/#tag) them by adding
some small indexation data to the output.

<AccountClient />

:::info Output Features

Different outputs can have different features. This table shows you which output has which features. You can find the
output specs in [TIP-018](https://wiki.iota.org/tips/tips/TIP-0018)

| Output  | Sender | Metadata | Immutable Metadata | Tag | Immutable Issuer |
| ------- | :----: | :------: | :----------------: | :-: | :--------------: |
| Basic   |   x    |    x     |                    |  x  |                  |
| Alias   |   x    |    x     |         x          |     |        x         |
| Foundry |        |    x     |         x          |     |                  |
| NFT     |   x    |    x     |         x          |  x  |        x         |

:::

## Example Code

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

The following example will:

Create multiple NFT outputs with all possible features using the
[`NftOutputBuilder`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/nft/struct.NftOutputBuilder.html)
and use the
[`NftOutputBuilder.add_feature`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/nft/struct.NftOutputBuilder.html#method.add_feature)
function to add the
[`SenderFeature`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/feature/struct.SenderFeature.html),
[`MetadataFeature`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/feature/struct.MetadataFeature.html)
and [`TagFeature`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/feature/struct.TagFeature.html)
as mutable features and use the
[`NftOutputBuilder.add_immutable_feature()`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/nft/struct.NftOutputBuilder.html#method.add_immutable_feature)
function to add the
[`IssuerFeature`](https://docs.rs/iota-sdk/latest/iota_sdk/types/block/output/feature/struct.IssuerFeature.html)
and `MetadataFeature` as immutable features.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/outputs/features.rs
```

</TabItem>
<TabItem value="typescript-node" label="Typescript (Node.js)">

The following example will:

Create multiple NFT outputs with all possible features using the
[`Client.buildNftOutput()`](../../references/nodejs/classes/Client/#buildnftoutput)
function and add the
[`SenderFeature`](../../references/nodejs/classes/SenderFeature/),
[`MetadataFeature`](../../references/nodejs/classes/MetadataFeature/)
and [`TagFeature`](../../references/nodejs/classes/TagFeature/)
as `features` and add the
[`IssuerFeature`](../../references/nodejs/classes/IssuerFeature/) and
`MetadataFeature` as `immutable_features`.

```typescript reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/nodejs/examples/how_tos/outputs/features.ts
```

</TabItem>
<TabItem value="python" label="Python">

The following example will:

Create multiple NFT outputs with all possible features using the
[`Client.build_nft_output()`](../../references/python/iota_sdk/client/#build_nft_output)
function and add the
[`SenderFeature`](../../references/python/iota_sdk/types/feature/#senderfeature-objects),
[`MetadataFeature`](../../references/python/iota_sdk/types/feature/#metadatafeature-objects)
and
[`TagFeature`](../../references/python/iota_sdk/types/feature/#tagfeature-objects)
as `features` and add the
[`IssuerFeature`](../../references/python/iota_sdk/types/feature/#issuerfeature-objects)
and `MetadataFeature` as `immutable_features`.

```python reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/python/examples/how_tos/outputs/features.py
```
