# Request Test Funds

You can use the [Shimmer faucet](https://faucet.testnet.shimmer.network/) to request test funds to develop and test your
application without incurring in any expenses. To do, you only need to input your address in and hit the request button.

Alternatively, you can follow any of these how-to guides to use your client software and language of choice to request
funds.
