
# Getting Started

:::tip Libraries Overview
You can find information on what client library suits your needs in
the [What is Shimmer? > Client Libraries in Shimmer](../../what-is-shimmer/libraries/client-libraries.md) Section
::::

You can use the following links to set up the client library of your choice:

## Wallet.rs

The [wallet.rs library](https://wiki.iota.org/shimmer/wallet.rs/getting_started/) is available in the following languages:

* [Rust](https://wiki.iota.org/shimmer/wallet.rs/getting_started/rust/)
* [Node.js](https://wiki.iota.org/shimmer/wallet.rs/getting_started/nodejs/)
* [Python](https://wiki.iota.org/shimmer/wallet.rs/getting_started/java/)
* [Java](https://wiki.iota.org/shimmer/wallet.rs/getting_started/python/)

## iota.rs

The [iota.rs library](https://wiki.iota.org/shimmer/iota.rs/getting_started/) is available in the following languages:

* [Rust](https://wiki.iota.org/shimmer/iota.rs/getting_started/rust/)
* [Node.js](https://wiki.iota.org/shimmer/iota.rs/getting_started/nodejs/)
* [Python](https://wiki.iota.org/shimmer/iota.rs/getting_started/java/)
* [Java](https://wiki.iota.org/shimmer/iota.rs/getting_started/python/)


## iota.js

You can install the [iota.js library](https://wiki.iota.org/shimmer/iotajs/getting_started/) using npm with a single command:

```bash
npm install @iota/iota.js@next
```
