## Code Example

The following example will:

1. Create a wallet.
2. Get Alice's account which was [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Send 1 SMR to an address using the funds owned by the alias. Make sure to update the code sample to send to the address of your choice.
